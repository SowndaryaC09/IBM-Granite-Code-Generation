# IBM Granite AI Code Generation Project

## Project Overview
This project demonstrates how IBM Granite AI can automatically generate Python code using prompt-based instructions.

The project was developed using Google Colab and Replicate API to run the IBM Granite model.

The generated application creates a simple bookstore interface called **Reader's Online Store**, displaying book titles and descriptions.

## Technologies Used
- Python
- IBM Granite AI
- Google Colab
- Replicate API
- Jupyter Notebook

## Features
- AI-generated Python code
- Prompt-based code generation
- Bookstore catalog interface
- Interactive widgets using ipywidgets

## How It Works
1. A prompt is given to IBM Granite AI.
2. Granite generates Python code automatically.
3. The code runs in Google Colab.
4. The output displays a bookstore catalog.
